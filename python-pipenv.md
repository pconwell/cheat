## pipenv

install pipenv
```
$ pip install pipenv
```

init pipenv for a project
```
$ cd my_project
$ pipenv install
```

install pacakges
```
$ pipenv install <package>
```

activate shell
```
$ pipenv shell
```

run script without activating shell
```
$ pipenv run python my_project.py
```

----

For new project:

1. git clone or create project directory
2. cd into directory
3. `$ pipenv install <dependencies>`
4. `$ pipenv shell`
5. run script
